      Acerca del proyecto

      ÓRBITA — Lista de tareas

      Aplicación web desarrollada para el curso de Aplicaciones y Programación Web. Integra HTML5 semántico, CSS con diseño espacial y JavaScript para gestión interactiva de tareas con almacenamiento local en el navegador.

      • Estructura HTML5 con navegación y 5 secciones
      • Formulario con validación en tiempo real
      • Mapa orbital animado y sistema de partículas
      • Modal de detalle, búsqueda, orden y cuenta regresiva
      • Logros desbloqueables y efectos al completar misiones
      • Persistencia con localStorage
