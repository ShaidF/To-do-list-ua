Basado en: Proyecto Grupal Final Fase 2 — Aplicaciones y Programación Web

───────────────────────────────────────────────────────────────
PASO 1 — Completar datos del grupo (carátula)
───────────────────────────────────────────────────────────────
1. Abre: Documentacion_Completa_ORBITA.html (doble clic).
2. Reemplaza SOLO los textos entre corchetes [ ]:
   • Carrera
   • Nombres de integrantes (1 a 5)
   • Grupo No., Semestre, Docente, Fecha
   • URLs de GitHub (sección 7)
   • Enlace del video (sección 9)

───────────────────────────────────────────────────────────────
PASO 2 — Publicar en GitHub Pages (OBLIGATORIO — 10%)
───────────────────────────────────────────────────────────────
1. Crear repositorio público en GitHub.
2. Subir TODA la carpeta WEB (index.html, css/, js/, data/, icons/, sw.js, etc.).
3. Settings → Pages → Deploy from branch → main → / (root).
4. Esperar URL: https://[usuario].github.io/[repositorio]/
5. Probar: crear misión → F5 → debe seguir visible.
6. Anotar URLs en sección 7 del documento.

───────────────────────────────────────────────────────────────

───────────────────────────────────────────────────────────────
PASO 4 — Video grupal (~5 minutos) (OBLIGATORIO — 10%)
───────────────────────────────────────────────────────────────
Grabar pantalla + voz (OBS, Zoom, etc.) incluyendo:

  PARTE GRUPAL:
  • Presentación del proyecto y objetivos
  • Demostración en funcionamiento (crear misión, completar, recargar)

  PARTE INDIVIDUAL (cada integrante explica UN tema):
  • HTML          → index.html, estructura y formulario
  • CSS           → styles.css, temas y responsive
  • JavaScript    → app.js, validación y eventos
  • localStorage  → DevTools + guardar/cargar datos
  • GitHub Pages  → repo, Pages y URL publicada

Ver guion detallado en Documentacion_Completa_ORBITA.html → Sección 9.
Subir video a YouTube (no listado) o Drive (enlace público) y pegar URL en sección 9.

───────────────────────────────────────────────────────────────
PASO 5 — Fichas individuales (una por integrante)
───────────────────────────────────────────────────────────────
1. Copiar Ficha_Individual_PLANTILLA.html por cada persona.
2. Renombrar: Ficha_Integrante_NombreApellido.html
3. Completar rol, archivos y guion de su parte del video.
4. Exportar cada ficha a PDF.

───────────────────────────────────────────────────────────────
PASO 6 — Exportar documento grupal a PDF
───────────────────────────────────────────────────────────────
Chrome o Edge → Abrir Documentacion_Completa_ORBITA.html
→ Ctrl + P → Guardar como PDF

───────────────────────────────────────────────────────────────

───────────────────────────────────────────────────────────────
